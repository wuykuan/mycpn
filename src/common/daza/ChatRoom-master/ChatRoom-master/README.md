项目地址：[仿微信聊天室](https://github.com/Jeffrey-0/ChatRoom)

## 效果图

![img](./public/images/聊天室.gif)

## 实现功能

- 在线群聊
- 发送表情
- 上传图片
- 私聊

## 快速开始

1. 安装项目

```shell
git clone git@github.com:Jeffrey-0/ChatRoom.git
```

2. 开启nodejs后台服务

```shell
node app.js
```

3. 浏览器访问

```shell
http://localhost:3000/
```
